# Trabalho de Estrutura de Dados com Ponteiros

### Sobre o projeto:
Este projeto foi desenvolvido como parte da disciplina Estrutura de Dados no IFRJ - Niterói, com o objetivo de aplicar os conceitos de ponteiros em Python.

---

### Estrutura orientada pelo professor:

O professor orientou que o projeto fosse dividido em quatro arquivos:
1. *`pilhaPonteiros.py`* - Pilha com Ponteiros 
    - Estrutura do tipo LIFO (Last In, First Out) - "Os últimos serão os primeiros"
  
2. *`filaPonteiros.py`* - Fila com Ponteiros
    - Estrutura do tipo FIFO (First In, First Out) - Primeiro que entra é o primeiro que sai
  
3. *`listaEncadeada.py`* - Lista Encadeada com Ponteiros
    - Cada nó aponta para o próximo
    
4. *`listaDumplamente,py`* - Lista Duplamente Encadeada
   - Cada nó aponta para o anterior e para o próximo 

---

### Como usar:

Basta rodar o arquivo Python da estrutura desejada. O programa já insere e remove os valores já incluídos e exibe sempre o estado atual da estrutura.

---

### Observação: 

Todo o código contém comentários explicativos, pois faz parte da minha forma de estudo. Eu escrevo comentários linha por linha para entender melhor o funcionamento interno da estrutura.
